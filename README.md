# 🏦 Smart Finance Assistant

## Project Overview

The Smart Finance Assistant is a Python-based finance analysis application developed for the ISYS2001 Final Programming Project.

This project helps users:
- Analyse transaction CSV files
- Track spending patterns
- Receive financial recommendations
- Ask finance-related questions
- Calculate savings goals

The application integrates:
- Python
- Pandas
- Gradio
- hands-on-ai concepts
- AI collaboration techniques

---

## Features

### 📊 CSV Spending Analysis
Users can upload a transaction CSV file containing:
- Date
- Amount
- Category
- Description

The assistant cleans and analyses the data to generate:
- Total spending
- Category summaries
- Spending percentages
- Financial recommendations

### 💬 Finance Chatbot
A finance-focused chatbot provides:
- Budgeting suggestions
- Spending advice
- Savings recommendations

### 🔍 RAG-Style Retrieval
The application includes a lightweight retrieval system that provides financial guidance from a small knowledge base.

### 🛠️ Savings Goal Calculator
Users can calculate how long it will take to reach a savings goal based on:
- Current savings
- Monthly contribution
- Target amount

### 🌐 Gradio User Interface
The application uses Gradio to provide a simple and interactive web interface.
